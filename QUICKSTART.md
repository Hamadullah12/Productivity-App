## Troubleshooting (If Something Goes Wrong)

### Error: "No devices found"
**Solution**: Start an emulator or connect a phone, or use Chrome:
```bash
flutter run -d chrome
```

### Error: "Hive not initialized" or similar
**Solution**: Clean and rebuild:
```bash
flutter clean
flutter pub get
flutter run
```

### Error: Package resolution failed
**Solution**: Update Flutter and retry:
```bash
flutter upgrade
flutter pub get
flutter run
```
