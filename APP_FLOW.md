## âœ… Key Design Decisions

1. **Hive over SQLite**: Faster, simpler, better for Flutter
2. **Provider over BLoC**: Easier to understand, sufficient for this app
3. **JSON serialization**: No code generation needed, more portable
4. **Local-first**: Works offline, privacy-focused
5. **Material 3**: Modern design, cross-platform consistency
6. **Single main timer**: Simpler state management for focus mode
7. **Dynamic Hive boxes**: Avoid build_runner complexity
8. **Emoji support**: Makes app friendly and motivational
